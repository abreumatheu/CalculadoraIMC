# Calculadora de IMC

A Calculadora de IMC foi o meu primeiro aplicativo, de desenvolvimento Android, atráves da linguagem Kotlin.
Foi um grande desafio, que foi superado com muita dedicação e estudos.

# Tecnologias
* Kotlin 
* Activity para host de telas
* ViewGroup
  * RelativeLayout
  * LinearLayout
  
* Components de UI
  * TextView
  * EditText
  * Button
* Toast para display de mensagens
* Intent Explicita para abrir a segunda Activity
* FindViewByld para recuperar components de UI
* String resource para concatenação de textos
